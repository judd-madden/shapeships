# Test Interfaces

This directory contains development and testing interfaces for Shapeships.

## Files

- **GameTestInterface.tsx** - Simplified 4-phase game test with multiplayer
- **FullPhaseTest.tsx** - Complete 14-subphase system test with ActionResolver
- **ActionResolverExample.tsx** - Example usage of ActionResolver hook
- **GameTestReadme.md** - Documentation for Game Test Interface

## Purpose

These interfaces are for development and testing only. They validate game logic, multiplayer synchronization, and phase systems before building the production UI.

Production game UI lives in `/game/display/`.
